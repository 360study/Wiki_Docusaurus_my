质量管理体系中的评价和评审:
1. 定义:
评价:对质量管理体系运行有效性与绩效的判断与分析。目的在于改进。
评审:对质量管理体系与相关标准要求的符合性进行检验。目的在于认证。
2. 内涵:
评价:范围广,涉及质量方针、目标、流程、资源等各要素。采用定量与定性相结合的方法。以业务目标为比较标准。结果为体系有效性的判断。
评审:范围窄,通常针对管理体系文件或某一流程。更加定性。以相关标准要求为评判依据。结果为证书颁发与否的推荐意见。
3. 区别:
(1)范围和对象不同:评价全面,评审局限;
(2)判断标准不同:评价以业务目标为标准,评审以相关标准为标准; 
(3)判断结果不同:评价以有效性为判断,评审以认证与否为判断;
(4)目的不同:评价以改进为目的,评审以认证证书为目的;
(5)方法不同:评价定量与定性相结合,评审更定性;
(6)使用对象不同:评价供内部使用,评审还面向外部;
(7)周期不同:评价周期长,评审周期定于相关标准。
4. 联系:
(1)同属于质量管理中的监督机制,目的在于确认管理体系运行的合理性;
(2)评价和评审的结果都可以为管理体系的优化提供依据,推动其不断发展完善; 
(3)评价的发现可以为认证评审做准备,证实体系的有效运行;评审可以 verificar 评价的结论;
(4)评价和评审都需要相关方的参与,加深其对管理体系的理解,建立互信关系。
综上,评价和评审在质量管理体系中有不同的定位和作用,但也存在一定的联系。组织可以根据管理需求和发展战略,选择适当的频次和方法进行评价与评审,彼此验证,共同促进质量管理体系的不断改进与发展。
