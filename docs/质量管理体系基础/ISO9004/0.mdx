> “适当的(appropriate)”与“适用的(applicable)”不得互换。“适当的”意指适合于或适于...
> 的,并意味着某种程度的自由;而“适用的”意指相关的或有可能应用的，且意味着如果能够做
> 到,就须要做。

<P>
  “考虑(consider)”一词意指有必要考虑这一话题，但可拒绝考虑;
  而“必须考虑count)"意指有必要考虑这一话题，但不能拒绝考虑。
</P>

<M>
  “持续(continual）”指发生在一段时期内的持续，但可能有间断;而“连续(continuous)”指不间
  断的持续，因此应当使用“持续"来描述改进。
</M>
<N>
  本标准中，“影响(effect）”一词用来描述组织变化的结果；“环境影响(environmental
  impaet)” 特指对环境造成变化的结果。
</N>

<O> “确保(ensure)”一词意指职责可以委派，但责任不能委派。</O>

<Q>
  短语“预期结果(intended
  outcome）”指组织通过实施其环境管理体系想要实现的结果。最低 限度的预期结果包括
  <A>提升环境绩效</A>、<B>履行合规义务</B>和<C>实现环境目标</C>
  。组织可针对其环境管理
  体系设定附加的预期结果，例如与其保护环境的承诺相一致，组织可建立一个致力于实现可持
  续发展的预期结果。
</Q>
