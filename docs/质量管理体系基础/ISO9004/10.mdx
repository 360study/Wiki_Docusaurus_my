### 10.1 总则

```
组织应确定改进的机会（见 9.1.9.2 和 9.3),并实施必要的措施，以实现其环境管理体系的预期
结果。
```

组织应确定改进的机会（包括环境绩效)以满足要求，并关注未来的霜求和期望;纠
正、预防或诚少不利影响;改进过程以防止不行合;改进管理体系的绩效和有效性。

組织采取措施改进时应当考虑风险和机過的确定、环境绩效分析和评价、合规性评
价、内部审核和管理评审的结果，应考虑投诉、不行合、事故、外部监管信息等。

改进可以是被动地(如纠正措施)、逐浙地（如持续改进)、跳跃变化（如突破性变更))、
创造性地（如创新)或重组（如转型)。

### 10.2 不符合和纠正措施

```
发生不符合时，组织应，
- a)对不符合做出响应，适用时：
  1. 采取措施控制并纠正不符合;
  2. 处理后果，包括减轻不利的环境影响，
- b)通过以下活动评价消除不符合原因的措施需求，以防止不符合再次发生成在其他地方发生，
  1. 评审不符合；
  2. 确定不符合的原因；
  3. 确定是否存在或是否可能发生类似的不符合，
- c)施任何所需的措施；
- d)评审采取的任何纠正指施的有效性；
- e)必要时，对环境管理体系进有交更。
措施应与所发生的不符台造成影啊(包括环境影响)的重要程度相适应。
组织应保留文件化信息作为下列事项的证据：
- 不行合的性质和所采取的任何后续措施；
- 任何纠正措施的结果。
```

#### (一)不符合的概念

凡是没有满足环境管理体系要求的情况均称为不符合，包括违反日常操作规定、监测
结果不符合法规要求、内审中的问题、管理评审中的问题等。

不符合的情况可包括：

- (1)体系绩效
  a)未建立环境目标和指标；
  b)未规定环境管理体系所要求的职责，如实现目标和指标的职责、或应急准备与响
  应的职责等；
  c)未定期实施合规性评价。
- (2)环境绩效
  a)未实现降低能耗的指标；
  b)未按规定的日程表进行维护；
  c)未能达到运行准则(如允许限值)。

#### (二)对不符合的响应

当发生不行合时，首先应对不符合做出响应，确定需采取的措施，包括控制并纠正不
符合、处理不符合产生的后果,减轻所造成的有害的环境影响，特别是出现紧急事故时更
应先及时采取措施解决问题，防止污染扩散。

#### (三)不符合的纠正措施和预防措施

除对不符合做出响应外，还应对不符合进行调查，分析引起不符合的原因，并及时采
取措施避免再次发生。

组织应识别是否存在或是否可能发生类似的不符合，评价是否需要采取预防措施，并
确定应采取的预防措施，以避免不行合发生。预防措施的要求包含在“4.1 理解组织及其
所处的环境”和“6.1 应对风险和机遇的措施”中。

纠正措施应与不符合发生后所造成后果的严重性及伴随的环境影响的程度相适应，
以避免产生小题大做或大题小做的现象。

#### (四)纠正措施的实施

管理者应当确保纠正措施得到实施。組织应规定相应的职责与权限，分级分层次实
施纠正措施。

应评审所采取的任何纠正措施的有效性,并采取系统的后续措施确保其有效性。

当所采取的措施需要对环境管理体系实施变更时，应当确保所有相关的文件、培训
和记录得到更新和批准,并使所有应当知道的人员知道这些变化。

#### (五)不符合的相关证据

组织应保留有关不符合的性质、所采取的任何后续指施及纠正措施的结果的记录，以
利于以后的追溯和(或)实施进一步的改进措施。这些记录也是管理评审的重要输人。

### 10.3 持续改进

```
组织应持续改进环境管理体系的适宜性、充分性与有效性，以提升环境缋效。
```

改进着眼于绩效、最终目的是提升环境绩效。改进是持续发生的.但不一定发生在体
系的每个方面，通过环境管理体系的整体运行或改进其一个或多个要素，都可能提升环
境绩效，

实现持续改进需要组织采取各种措施、例如配备必要的人力、保障财务或技术资源、
商定并运行相关流程、实施激励制度等，组织应当确定所采取的措施及其等级、程度与时
间表，并确保这些措施得以实施。
