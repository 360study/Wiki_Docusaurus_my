环境管理体系中的应急准备和响应可以理解为:
1. 组织针对环境事故或突发事件制定的一系列应对措施,包括事前的预防与准备、事中的应急响应、事后善后与改进。
2. 一种管理流程,包括环境监测、风险评估、应急预案制定、资源准备、培训演练、事故发生时的紧急决策与控制、善后治理等环节。
3. 组织保护环境与应对环境事件变化的战略与能力。它体现了组织在环境管理上的觉悟与水平。
4. 组织管理层保障环境安全与应变能力的重要手段。它使组织能够在环境事件发生时迅速作出响应,最大限度减少损失,改善环境影响。
5. 组织与利益相关方之间在环境事件中相互配合与协作的机制。它需要组织与政府、公众、媒体等建立起信息沟通与联动机制。
6. 组织不断提高环境管理水平的动力与依据。通过总结应急准备与响应的经验,可以发现环境管理体系中的不足,并采取改进措施。
7. 组织履行社会责任的重要体现。良好的应急准备与响应可以最大限度地避免或减少环境事故对社会的危害,显示出组织高度的社会责任心。
综上,应急准备与响应在环境管理体系中起着极为重要的作用。它使组织的环境管理实现从被动应对到主动预防的转变,成为组织实现可持续发展的关键所在。组织应将其纳入战略管理,并持续改进,以适应环境变化。
